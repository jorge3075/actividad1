1. ¿Qué es un Sistema Operativo?

Es un intermediario entre el usuario y el hardware de la PC que permite usar los programas instalados en la máquina.

2. ¿Componentes de un Sistema Operativo?

Sus componentes son: el kernel o núcleo, interfaz de usuario, drivers, sistemas de archivos, sistema de protección, sistemas E/S, gestión de procesos, memoria principal, sistemas de comunicaciones, gestor de recursos.

3. ¿Tareas del Sistema Operativo?

Gestionar la memoria de acceso aleatorio y ejecutar las aplicaciones, designando los recursos necesarios

4. Funciones de un Sistema Operativo

Gestionar la memoria de acceso aleatorio y ejecutar las aplicaciones, designando los recursos necesarios

5. Características de un Sistema Operativo

manejo de recursos, control de operaciones del sistema, recuperación de errores y manejo de memoria.

6. Tipos de Sistema Operativo:

• Monotarea: solo puede ejecutar una tarea o programa a la vez. • Monousuario: es el sistema operativo que solo puede responder a un usuario a la vez. • Multitareas: son aquellos que permiten que varios programas se ejecuten en el mismo momento en uno o varios ordenadores a la vez. • Multiprocesador: hace posible que un mismo programa sea utilizado en más de un ordenador.

21. Qué es un proceso en entorno de los sistemas operativos

es una entidad fundamental que representa una tarea o programa en ejecución en un programa informático. Cada proceso tiene su propio espacio de memoria y recursos asignados, lo que permite la multitarea y la gestión eficiente de los recursos del sistema.

Qué es un sistema operativo distribuido y centralizado:

 Centralizado: es aquel que solo permite utilizar los recursos de un solo ordenador.  Distribuido: es aquel que permite ejecutar los procesos de más de un ordenador al mismo tiempo.

• FAT (File Allocation Table o tabla de asignación de archivos) • Ex FAT (Extended File Allocation Table o tabla de asignación de archivos extendida) • NTFS (New Technology File System) • HFS+ (Hierarchical File System) • APFS (Apple File System)
